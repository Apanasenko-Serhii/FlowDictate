# FlowDictate — спека v1 (затверджено користувачем 2026-07-04)

Локальний аналог Wispr Flow: системна диктовка у будь-яке вікно Windows,
повністю офлайн, без лімітів. Двигун — faster-whisper на GPU (RTX 4060 Ti).

## Затверджені рішення (інтерв'ю 2026-07-04)
1. **Режим клавіші:** push-to-talk — тримаєш клавішу/комбо, говориш; відпустив → текст друкується.
   Клавіша за замовчуванням: `right ctrl`. Підтримка КОМБІНАЦІЙ (2026-07-04):
   `ctrl+shift`, `ctrl+alt+space` тощо — старт коли натиснуті всі, стоп коли відпущено будь-яку.
   Реалізація — клас HotkeyListener; unit-тест машини станів hotkey_test.py — 10/10 PASS.
2. **GPU:** розблокувати через pip-пакети `nvidia-cublas-cu12` + `nvidia-cudnn-cu12`
   (DLL підключаються в рантаймі через os.add_dll_directory). Фолбек — CPU int8 автоматично.
3. **Вставка:** одразу в активне вікно (як Wispr Flow) — через буфер обміну + Ctrl+V,
   з відновленням попереднього вмісту буфера.
4. **Пакування:** PyInstaller onedir → `dist/FlowDictate/FlowDictate.exe` (вимога користувача).

## Технічний склад
- Python 3.11 (системний), окремий venv у корені проєкту.
- Захоплення звуку: `sounddevice`, 16 kHz mono float32.
- Гаряча клавіша: `keyboard` (глобальний хук Windows).
- ASR: `faster-whisper`, модель `large-v3-turbo` (GPU float16), фолбек CPU int8.
  Мова: авто (укр/рос/англ), словник домену через initial_prompt
  (меблева термінологія: ЛДСП, кромка, Блюм, Базіс...).
- Вставка: pywin32 clipboard + SendInput Ctrl+V.
- Трей: `pystray` + `Pillow` (стан: сірий=спить, червоний=запис, жовтий=транскрибує).
- Конфіг: `config.json` поруч з exe (hotkey, model, language, dictionary).
- Логи: `flowdictate.log` поруч з exe.
- Модель НЕ пакується в exe — качається у %LOCALAPPDATA%\FlowDictate\models при першому запуску.

## Чим кращий за Wispr Flow
- Безлімітно й безкоштовно; аудіо не покидає ПК; свій словник під меблевий домен.
- Гачок на майбутнє: командний режим у шину агентів (v2, не в цій ітерації).
