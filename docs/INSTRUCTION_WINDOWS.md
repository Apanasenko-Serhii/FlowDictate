# FlowDictate — інструкція (Windows)

Локальна диктовка «як Wispr Flow, тільки своя»: тримаєш клавішу → говориш →
відпускаєш → текст друкується в активне вікно. Повністю офлайн, без лімітів
слів, аудіо не покидає комп'ютер.

> Статус: зібрано й перевірено на цьому ПК (RTX 4060 Ti). Живий тест
> мікрофоном — за користувачем.

---

## 1. Що потрібно (вимоги)

- Windows 10 / 11 (64-bit).
- Мікрофон.
- Для миттєвої швидкості — відеокарта NVIDIA. Без неї працює теж, але
  розпізнавання йде на процесорі й помітно повільніше.
- ~3 ГБ вільного місця (сам застосунок + модель розпізнавання).
- Інтернет — лише **один раз**, під час першого запуску (качається модель).
  Далі все офлайн.

## 2. Встановлення

1. Скопіюй усю папку `FlowDictate` (з `dist`) у зручне місце, напр.
   `C:\Tools\FlowDictate`. Папку переносимо **цілком** — не вихоплюй один exe.
2. Подвійний клік по `FlowDictate.exe`.
3. У треї (біля годинника) з'явиться круглий значок:
   - **сірий** — вантажиться модель. Перший раз качає ~1.6 ГБ, це кілька
     хвилин; надалі — 3–4 секунди.
   - **зелений** — готовий до роботи.

Модель зберігається у `C:\Users\<ти>\AppData\Local\FlowDictate\models` —
качається раз і назавжди.

## 3. Як користуватись

1. Постав курсор туди, куди хочеш надиктувати текст (Telegram, Word,
   браузер, поле пошуку — будь-де).
2. **Затисни й тримай** гарячу клавішу (за замовчуванням — **правий Ctrl**).
   Значок стане **червоним** — іде запис.
3. Говори.
4. **Відпусти** клавішу. Значок стане **жовтим** (розпізнаю) → за мить
   текст сам надрукується там, де курсор.

Звук підказує: короткий вищий «біп» — запис почався; нижчий «біп» — текст
вставлено. Значок повернувся в зелений — можна диктувати далі.

**Вихід:** правою кнопкою по значку в треї → «Вихід».

## 4. Налаштування (`config.json` поруч з exe)

Відкрий `config.json` у Блокноті, зміни значення, збережи, перезапусти exe.

| Поле | Що робить | Приклади |
|------|-----------|----------|
| `hotkey` | Клавіша або комбінація push-to-talk | `right ctrl`, `f9`, `caps lock`, `ctrl+shift`, `ctrl+alt+space`, `alt+space` |
| `language` | Мова диктовки | `auto` (сам визначає), `uk`, `en`, `ru` |
| `model` | Модель розпізнавання | `large-v3-turbo` (дефолт, найкраща), `medium`, `small` (для слабких ПК) |
| `initial_prompt` | Словник термінів та імен — покращує розпізнавання рідкісних слів | напр. перелічити прізвища клієнтів, назви матеріалів |
| `sounds` | Звукові підтвердження | `true` / `false` |

**Про комбінації:** для комбо (`ctrl+shift` тощо) запис починається, коли
натиснуті **всі** клавіші, і зупиняється, щойно відпущено **будь-яку** з них.

## 5. Якщо щось не так

- **Значок сірий і не зеленіє довго** — при першому запуску якраз качається
  модель; дай кілька хвилин. Якщо застрягло надовго — перевір інтернет і
  перезапусти (недокачана модель докачається сама).
- **Текст не вставляється в певну програму** — Windows не дозволяє вставляти
  у вікна, запущені **від імені адміністратора**. Або запусти FlowDictate
  теж від адміністратора, або диктуй у звичайні вікна.
- **Нічого не чути / порожній текст** — перевір, що говориш у правильний
  мікрофон (системні налаштування звуку → пристрій вводу за замовчуванням).
- **Що сталося — дивись у** `flowdictate.log` поруч з exe (там усі події й
  помилки з часом).

## 6. Автозапуск разом з Windows (за бажанням)

Натисни `Win+R`, введи `shell:startup`, Enter — відкриється папка автозапуску.
Створи там ярлик на `FlowDictate.exe`. Тепер застосунок стартуватиме сам при
вході в систему.
