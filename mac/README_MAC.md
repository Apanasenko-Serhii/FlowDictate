# FlowDictate для macOS — вихідний код + збірка

Це **вихідний пакет** Mac-версії. Готового `.app` тут ще немає — його треба
зібрати **на самому Mac** (з Windows зібрати macOS-застосунок неможливо).
Ядро розпізнавання те саме, що на Windows; системні частини — Mac-нативні.

## Два способи запуску

### Спосіб 1 — швидкий старт без збірки (рекомендую спершу)
1. Розпакуй цю папку кудись у Home, напр. `~/FlowDictate`.
2. Постав базові інструменти (один раз), у Terminal:
   ```bash
   brew install python@3.11 portaudio ffmpeg
   ```
3. Запусти:
   ```bash
   cd ~/FlowDictate
   bash install_mac.command
   ```
   Перший запуск поставить залежності (~1 ГБ) і стартує застосунок.
4. Дай дозволи (див. нижче) і перезапусти.

### Спосіб 2 — зібрати клікабельний `.app`
```bash
cd ~/FlowDictate
bash build_mac.sh
```
Отримаєш `dist/FlowDictate.app` — перетягни в `/Applications`.

## Обов'язкові дозволи macOS
**System Settings → Privacy & Security**, додай застосунок (Terminal для
способу 1, або FlowDictate.app для способу 2) у ТРИ списки:
- **Microphone** — чути голос,
- **Accessibility** — друкувати текст у чужі вікна,
- **Input Monitoring** — ловити гарячу клавішу глобально.

Після додавання — перезапусти застосунок.

## Використання
Тримай гарячу клавішу (за замовчуванням **right Option ⌥**) → говори →
відпусти → текст вставиться. Індикатор стану — значок у меню-барі вгорі:
◌ вантажиться · 🟢 готовий · 🔴 запис · 🟡 розпізнаю.

Налаштування — `config.json` (створюється поруч): `hotkey` (напр. `right alt`,
`cmd+shift`, `ctrl+alt+space`), `language` (`auto`/`uk`/`en`), `model`,
`initial_prompt` (словник термінів).

## Швидкість — важливо
На Mac двигун працює **на процесорі** (Metal у ньому не підтримується). На
Apple Silicon (M1–M4) це прийнятно, але не так миттєво, як на NVIDIA. Якщо
захочеш максимум швидкості — є план перевести двигун на whisper.cpp з Metal
(див. `docs/INSTRUCTION_MAC.md`, шлях Б).

  за тобою** — я не мав Mac під рукою. Якщо щось не так — надішли
  `~/Library/Application Support/FlowDictate/flowdictate.log`, поправлю.

## Файли в пакеті
- `flowdictate_mac.py` — застосунок.
- `install_mac.command` — швидкий старт без збірки.
- `build_mac.sh` + `setup_mac.py` — збірка `.app` через py2app.
- `mac_hotkey_test.py` — автотест логіки клавіш.
- `config.example.json` — приклад конфіга.
