# FlowDictate v2 — спека «краще за Wispr» (чернетка на затвердження)

Ціль: найкраща транскрипція + «топ»-відчуття, офлайн. Рішення інтерв'ю 2026-07-04.

## Затверджені рішення користувача
- **AI-чистка:** через **Ollama** (уже працює локально; підтягнути малу модель).
- **Стиль чистки:** **Легко** — прибрати «еее/ну», пунктуація, великі літери; слова НЕ переписувати.
- **Топ-фічі (усі 4):** менеджер словника, прев'ю перед вставкою, історія надиктовок, голосова пунктуація.
- **CPU-lite збірка** для тих, у кого немає NVIDIA (окремий менший архів).
- **Іконка:** логотип користувача замість кольорового кружка (трей + вікно + exe).

## Конвеєр v2 (порядок обробки)
```
запис → whisper (тюнінг + словник) → голосова пунктуація/команди
      → [AI-чистка Ollama, легка] (опц.) → [прев'ю] (опц.) → вставка + історія
```

## 1. Точність whisper (без нових залежностей)
- Модель за замовч. лишаємо `large-v3-turbo`; у налаштуваннях — вибір `large-v3` (максимум точності).
- Тюнінг: `beam_size` 5, `temperature` fallback, `condition_on_previous_text=False`, VAD-параметри
  (менше обрізань пауз), `no_speech_threshold`. Мета — менше пропусків і «злипань».
- initial_prompt формується зі **словника** (нижче).

## 2. AI-чистка (Ollama, локально, опційно)
- Двигун: Ollama (`http://localhost:11434`), OpenAI-сумісний ендпоінт. Модель: **qwen2.5:3b-instruct**
  (~2 ГБ, сильна в укр., швидка) — я підтягну `ollama pull`. Ендпоінт/модель — у config.
- Промпт «легкий»: прибрати паразити + пунктуація/регістр, **не змінювати зміст і слова**.
- Гард: якщо Ollama недоступна або таймаут → вставляємо сиру транскрипцію (не блокуємось). Тумблер on/off.

## 3. Менеджер словника (у вікні налаштувань)
- Багаторядковий список імен/термінів/брендів. Зберігається в config, іде у whisper `initial_prompt`
  і (за наявності) як підказка в AI-чистку. Приклад: прізвища клієнтів, «Blum», «ЛДСП».

## 4. Голосова пунктуація / команди (post-whisper)
- Заміни: «кома»→«,», «крапка»→«.», «знак питання»→«?», «з нового рядка»→перенос, «новий абзац»→2 переноси.
- Команди: «видали останнє» (стерти попередню вставку). Тумблер on/off (щоб не заважало звичайній диктовці).

## 5. Прев'ю перед вставкою (опційно, тумблер)
- Маленьке вікно з розпізнаним (і почищеним) текстом: Enter/кнопка «Вставити», Esc — скасувати,
  можна поправити руками перед друком. Вимкнено за замовч. (щоб не сповільнювати швидкий сценарій).

## 6. Історія надиктовок
- Останні N (напр. 50) результатів у пам'яті/файлі. Пункт у треї «Історія» → список, клік = скопіювати
  в буфер / вставити повторно.

## 7. Іконка = логотип користувача
- Трей-іконка, іконка вікна налаштувань і вбудована іконка exe/.app. Стани (готовий/запис/зайнято)
  показуємо накладенням кольорової крапки на лого (щоб статус лишився видимим).
- **ВІДКРИТЕ:** потрібен файл лого (PNG з прозорістю або SVG/ICO, бажано квадратний). Джерело — за тобою.

## 8. CPU-lite збірка (Windows, для не-NVIDIA)
- Той самий код, PyInstaller БЕЗ CUDA-DLL → ~300–500 МБ замість 1.6 ГБ. Пристрій авто-`cpu int8`.
- Окремий архів `FlowDictate-Windows-cpu.zip` у релізі. GPU-версія лишається для NVIDIA.

## Verify перед здачею (мій план перевірки)
- Точність/чистка: прогін тестових wav → whisper + Ollama-чистка, звірка тексту.
- Голосова пунктуація: unit-тест замін.
- Словник/прев'ю/історія: smoke (config round-trip, вікна будуються).
- CPU-lite: зібрати, запустити, переконатись `device=cpu`, Ready.
- Живий тест мікрофоном + вигляд лого — за користувачем.
- Оновити exe (GPU + CPU-lite), перезалити реліз + хмарну Mac-збірку.
